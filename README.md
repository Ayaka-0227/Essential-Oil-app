# Essential-Oil-app
##サービス概要
現状の精神状態・身体状況に合ったアロマオイルの種類をご提案してくれるアプリです。
本人にメンタルチェックを行ってもらうことで、上記の状態を把握し、チャート化して伝えます。
その後、アプリが当事者にお勧めのアロマオイルを提案し、説明させていただきます。

##このサービスへの思い・作りたい理由
私は以前、アロマオイルを使っていた全身で行っていました。
そこはアロマオイルが数100種類あるのにも関わらず、主に使用されていたアロマオイルは20種類ほどだけでした。
こんなにも多くのアロマオイルをコレクションされているのに、使用されていないアロマオイルは沢山あり、もったいないと感じました。
そんな時にメンタルチェックをして、偏りのないようにお客様に正しいものを提供できたら無駄になるアロマオイルもなくなるのではないかと思い、このようなサービスを考えてみました。

##ユーザー層について
主に30代～50代です。
20代は比較的、社会的なストレスが少なく、アロマオイルに触れる機会は少ないと考えました。
30代～50代はライフスタイル職場での立場などで大きな変化があり、ストレスを抱えている方が多く、アロマオイルを身近に使っている方が多いと感じたからです。

##サービス利用イメージ
正しいアロマオイルを使うことでストレスの解消、リラックス効果が生まれます。

##ユーザーの獲得について
ユーザー登録を行っていただき、ユーザーごとに異なるサービスを提供します。
メンタルチェックの結果は五角形レーダーチャートで表示され、当事者の現状を教えてくれます。

##サービスの差別化ポイント・推しポイント
アロマオイルの効果を種類別に提案しているサービスはありますが、メンタルチェックを行い、当事者に応じたアロマオイルをお勧めしてくれるアプリはこのアプリだけです！

##機能候補
-メンタルチェックシート
  メンタルチェックと表示されたボタンの下にこのチェックは医療的な診断を行うものではありませんと注意書きをします。
  ボタンを押していただくことチェックシートへ移り、質問が提示されます。
  回答の平均時間は3分であり、すべて回答を終えることで結果が表示されます。
  質問は合計8問、回答は各5段階、過去7日間を基準にします。
-五角形レーダーチャート
  ストレス、疲労、不安、睡眠不足、活力不足の項目で表示されます。
-ユーザー登録
-ログイン・ログアウト機能
-管理者画面
  アロマオイルのCRUD機能、ユーザー管理・検索機能、ストロングパラメータ


##使う技術スタック
-利用するフレームワーク（Ruby on Rails）

-使用予定のライブラリ(CSS, JavaScript, Turbo)
