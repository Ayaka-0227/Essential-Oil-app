# Essential-Oil-app
## サービス概要
現状の精神状態・身体状況に合わせたアロマオイルの種類を提案をしてくれるアプリです。
当事者にメンタルチェックを行っていただくことで、上記の状態を把握し、チャート化にして教えてくれます。
その後、アプリが当事者にお勧めのアロマオイルを提案し、説明をしてくれます。

## このサービスへの思い・作りたい理由
私は以前、アロマオイルを用いた整体で勤めていました。
そこではアロマオイルが数100種類あるのにも関わらず、主に使用されていたアロマオイルは20種類ほどだけでした。
こんなにも多くのアロマオイルをコレクションされているのに、使用されていないアロマオイルは数多くあり、勿体無いと感じました。
そんな時にメンタルチェックをして、偏りのないようお客様に適切なものを提供できたら無駄になるアロマオイルもなくなるのではないかと思い、このようなサービスを作ろうと考えました。

## ユーザー層について
主に30代～50代です。
20代は比較的、社会的ストレスが少なく、アロマオイルに触れ合う機会は少ないと考えました。
30代～50代はライフスタイルや職場での立場などで大きな変化があり、ストレスを抱えている方が多く、アロマオイルを身近に使用している方が多いと感じたからです。

## サービスの利用イメージ
適切なアロマオイルを使用することでストレスの解消、リラックス効果が生まれると思います。

## ユーザーの獲得について
メンタルチェックした結果は五角形レーダーチャートで表示され、どのような状態か教えてくれます。

## サービスの差別化ポイント・推しポイント
アロマオイルの効果を種類別に提示しているサービスはありますが、メンタルチェックを行い、当事者に応じたアロマオイルをお勧めしてくれるアプリはこのアプリだけです！

## 機能候補
メンタルチェックの心理テスト、五角形レーダーチャート

## 使用する技術スタック
- 使用するフレームワーク（Ruby on Rails）
- データベース（PostgreSQL）
- デプロイ先（Render）
- 使用予定のライブラリ(CSS, JavaScript, Turbo)
